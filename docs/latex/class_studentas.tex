\doxysubsection{Detailed Description}
Student class that inherits from \doxylink{class_zmogus}{Zmogus} (Person) abstract class. 

This class represents a student with homework grades, exam score and final grade. It inherits from the abstract \doxylink{class_zmogus}{Zmogus} class and implements its pure virtual methods.
